## Project Overview :
A full product dissection of Rapido, India's #1 bike-taxi aggregator,
along with a 7-entity relational database schema design.

## Skills Demonstrated :
- SQL
- ER Diagram Design
- Product Thinking
- Real-World Problem Analysis

## Key Highlights :
- Identified 4 core real-world problems Rapido solves
- Designed 7 entities: Users, Captains, Rides, Payments, Ratings, Promos
- Rides is the central hub — 5 of 6 relationships pass through it
